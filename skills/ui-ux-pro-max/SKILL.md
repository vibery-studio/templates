# UI/UX Pro Max - Design Intelligence

Comprehensive UI/UX design intelligence with searchable database covering 50 styles, 21 palettes, 50 font pairings, 20 charts, and 8 development stacks.

## When to Use

- Building beautiful, stunning, or aesthetic interfaces
- Making design decisions before implementation
- Selecting styles, colors, typography, or chart types
- Creating landing pages, dashboards, or mobile apps
- Implementing responsive designs

## Core Workflow

1. Analyze requirements
2. Search relevant domains (product, style, typography, color, landing, chart, UX, stack)
3. Implement using stack-specific guidelines

**Default Stack:** HTML with Tailwind CSS

**Alternatives:** React, Next.js, Vue, Svelte, SwiftUI, React Native, Flutter

## Search Domains

- **Product Types:** Recommendations for app categories
- **UI Styles:** Glassmorphism, claymorphism, minimalism, brutalism, neumorphism, bento grid, dark mode
- **Font Pairings:** 50 curated Google Fonts combinations
- **Color Palettes:** Industry-specific palettes (SaaS, healthcare, fintech)
- **Landing Pages:** Structure templates and patterns
- **Chart Types:** 20 chart types with library recommendations
- **UX Best Practices:** 98 guidelines for accessibility and usability

## Professional Standards

### Visual Elements
- Use SVG icons (Heroicons, Lucide, Simple Icons) - NO emoji icons
- Maintain consistent sizing
- Verify brand logos from official sources

### Interaction Design
- All interactive elements require `cursor-pointer`
- Hover states provide feedback without layout shifts
- Transitions use 150-300ms timing

### Accessibility & Contrast
- Light mode text: 4.5:1 minimum contrast
- Glass elements must be visible
- Borders visible in both light and dark modes

### Layout Standards
- Floating navbars need edge spacing
- Content shouldn't hide behind fixed elements
- Test responsive design at 320px through 1440px

## Pre-Delivery Verification

- [ ] Visual quality verified
- [ ] Interactions work correctly
- [ ] Theming consistent (light/dark)
- [ ] Layout responsive
- [ ] Accessibility checked
